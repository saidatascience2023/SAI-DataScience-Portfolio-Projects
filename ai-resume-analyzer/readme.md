
# AI Resume Analyzer

An interactive Streamlit app that analyzes a resume against AI and Data career roles.

## Features

- Upload resume
- Select target role
- Get resume match score
- Identify missing skills
- Get project suggestions
- Prepare interview questions

## Roles Covered

- Data Analyst
- Data Scientist
- Machine Learning Engineer
- Generative AI Engineer
