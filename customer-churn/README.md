# Customer Churn Prediction App

A Streamlit portfolio project that predicts customer churn risk using a Random Forest Classifier.

## Features
- Customer churn prediction
- Random Forest model
- Churn probability score
- Feature importance chart
- Retention action suggestions
- Streamlit deployment-ready

## Run Locally

```bash
pip install -r requirements.txt
streamlit run app.py
```

## Project Skills
- Machine Learning
- Classification
- Customer Analytics
- Feature Engineering
- Model Evaluation
- Streamlit Deployment
